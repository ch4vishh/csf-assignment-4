#Calculator Project (CSF Capstone)

This mini calculator project is created for **Assignment 04: Tools for Programming, Learning, and Collaboration** under the course **Computer Science Fundamentals & Career Pathways**.

It performs simple arithmetic operations using Python.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

#Project Description
This Python calculator can do:

- Addition  
- Subtraction  
- Multiplication  
- Division  

The project uses:
- Visual Studio Code  
- Git (Version Control)  
- GitHub (Hosting the project)  

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

#How to Run the Project

1. Clone the repository
2.  Run the Python file

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

#Features
- Simple and clean console interface  
- Handles divide-by-zero errors  
- 5 meaningful Git commits  
- Easy to understand functions
